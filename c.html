What is the difference between "int" and "float" data types in C?

The "int" data type is used to store integer values, while the "float" data type is used to store floating-point (decimal) values. Integers have no fractional part, while floating-point numbers can have decimal places.

Explain the difference between "while" and "do-while" loops.

Both "while" and "do-while" loops are used for repetitive execution. The key difference is that the "while" loop checks the condition before executing the loop body, while the "do-while" loop executes the loop body at least once before checking the condition.

What is the purpose of the "break" statement in C?

The "break" statement is used to terminate the current loop or switch statement and resume execution after the loop or switch.

How can you allocate memory dynamically in C?

You can allocate memory dynamically using the malloc() function. For example: int *ptr = (int *)malloc(sizeof(int));

Explain the difference between "strcpy" and "strncpy" functions.

Both functions are used to copy strings, but "strcpy" copies characters until it encounters a null terminator ('\0'), potentially causing buffer overflows. "strncpy" allows you to specify a maximum number of characters to copy, reducing the risk of buffer overflow.

What are pointers in C?

Pointers are variables that store memory addresses. They allow you to indirectly access and manipulate data stored in memory.

How do you pass arguments to a function in C?

Function arguments can be passed by value or by reference (using pointers). Passing by value involves making a copy of the value, while passing by reference allows the function to modify the original data.

What is the "sizeof" operator used for?

The "sizeof" operator returns the size, in bytes, of a data type or variable. It's often used to allocate memory dynamically or to calculate the size of data structures.

Explain the concept of recursion in C.

Recursion is a technique where a function calls itself to solve a problem. It involves a base case that terminates the recursion and one or more recursive cases that break down the problem into smaller subproblems.

How are arrays and pointers related in C?

Arrays and pointers are closely related. An array name can be used as a pointer to its first element, and pointer arithmetic can be used to access other elements in the array.

What is the difference between "scanf" and "printf" functions?

scanf is used to read input from the user, and printf is used to print output to the screen. They are complementary functions for input and output operations.

Explain the difference between "static" and "dynamic" memory allocation.

Static memory allocation is done at compile time, and the memory size is fixed. Dynamic memory allocation is done at runtime using functions like malloc, calloc, and realloc, and the memory size can change as needed.

What is a "const" pointer?

A const pointer is a pointer that points to a constant value, meaning you can't modify the value through that pointer. However, the pointer itself can be changed to point to another memory location.

Explain the difference between "calloc" and "malloc".

Both calloc and malloc are used for dynamic memory allocation. The key difference is that calloc initializes the allocated memory to zero, while malloc leaves the memory uninitialized.

How does the "switch" statement work?

The switch statement is used to evaluate an expression against multiple possible cases. It transfers control to the case that matches the expression and continues until a break statement is encountered.

What are the differences between "structure" and "union"?

Structures allow you to group different data types under one name, while unions allow you to store different data types at the same memory location. However, only one member of the union can hold a value at a time.

Explain the concept of "bitwise operators".

Bitwise operators operate on individual bits of a value. Common bitwise operators include AND (&), OR (|), XOR (^), left shift (<<), and right shift (>>).

What is a "null pointer" and how can it be used safely?

A null pointer is a pointer that doesn't point to any valid memory location. It's often used to indicate the absence of a valid value or to initialize pointers before they point to valid memory addresses.

Describe the process of "function pointer" usage in C.

Function pointers allow you to store the address of a function and call that function through the pointer. They provide flexibility by allowing you to choose which function to call at runtime.

Explain the difference between "pre-increment" and "post-increment" operators.

The "pre-increment" (`++i`) operator increments the value before using it, while the "post-increment" (`i++`) operator uses the value and then increments it.

How can you reverse a string in C?

You can reverse a string by swapping characters from the start and end of the string, moving towards the center.

void reverseString(char *str)
{
int length = strlen(str);
for (int i = 0; i < length / 2; i++) {
char temp = str[i];
str[i] = str[length - 1 - i];
str[length - 1 - i] = temp;
}
}

Explain the concept of "recursion termination" in a recursive function.

Recursion termination involves defining a base case that stops the recursive calls. Without a base case, the recursion would continue indefinitely, causing a stack overflow.

How do you define a macro in C?

A macro is defined using the `#define` directive. Macros are typically used for simple text substitution.

Explain the difference between "strcpy" and "memcpy".

Both functions copy memory, but `strcpy` is specific to strings and stops at the null terminator, while `memcpy` copies a specified number of bytes and isn't restricted to strings.

What is the "volatile" keyword used for?

The "volatile" keyword indicates that a variable's value may change unexpectedly, typically due to external factors, and should not be optimized by the compiler.

How can you find the length of a null-terminated string using a pointer?

You can iterate through the string using a pointer and count the characters until you reach the null terminator.

int stringLength(const char *str) {
int length = 0;
while (*str != '\0') { length++;
str++;
}
return length;
}
