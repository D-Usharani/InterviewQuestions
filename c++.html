What is C++?

C++ is a general-purpose programming language that extends the capabilities of the C programming language. It supports both procedural and object-oriented programming paradigms, making it suitable for a wide range of applications.

What is the difference between C and C++?

C++ is an extension of the C programming language with added features like classes, objects, inheritance, and polymorphism. C++ supports both procedural and object-oriented programming, whereas C is primarily procedural.

What are the four pillars of object-oriented programming?

The four pillars of object-oriented programming are:

  • Encapsulation: Bundling data and methods that operate on the data into a single unit (class).
  • Inheritance: Creating a new class from an existing class, inheriting its properties and behaviors.
  • Polymorphism: The ability of objects of different classes to be treated as objects of a common base class.
  • Abstraction: Representing complex real-world entities with simplified models.

What is the difference between "class" and "struct" in C++?

In C++, the primary difference between "class" and "struct" is that the default access level is "private" for class members and "public" for struct members. Additionally, classes support encapsulation by providing private and protected access specifiers.

Explain the concept of "operator overloading" in C++.

Operator overloading allows you to define custom behaviors for operators when applied to user-defined classes or structs. For example, you can define how the "+" operator works for adding instances of your class.

What is the difference between a class and an object?

A class is a blueprint or template for creating objects. It defines the properties (data members) and behaviors (member functions) that objects of that class will have. An object is an instance of a class, created based on the class blueprint.

What is constructor overloading?

Constructor overloading is the ability to define multiple constructors within a class, each with a different set of parameters. This allows objects to be initialized in various ways, providing flexibility and convenience.

What is a destructor?

A destructor is a special member function in a class that is called when an object is destroyed or goes out of scope. It is used to release any resources acquired by the object, such as memory or file handles.

What is operator overloading?

Operator overloading allows you to define custom behaviors for operators such as +, -, *, /, etc., when used with objects of a class. It enables objects to mimic the behavior of built-in data types.

What is the difference between "new" and "malloc"?

"new" is an operator in C++ used to allocate memory for objects dynamically. It calls the constructor and returns a pointer to the allocated memory. "malloc" is a function in C used to allocate memory, and it returns a void pointer. "new" is type-safe and allows object creation, while "malloc" is not type-safe and requires manual memory management.

Explain the concept of virtual functions and dynamic polymorphism.

Virtual functions are functions declared in a base class with the "virtual" keyword. They enable dynamic polymorphism, allowing a derived class to override the implementation of the base class's virtual function. The appropriate function is determined at runtime based on the object's actual type, providing flexibility and extensibility.

What is the "const" keyword in C++?

The "const" keyword is used to indicate that a variable, parameter, or member function does not modify the value it references. For variables, it ensures immutability. For member functions, it indicates that the function will not modify the object's state.

What is a template in C++?

A template in C++ is a powerful feature that allows you to create generic classes or functions that work with different data types without having to rewrite the code for each type. Templates provide a way to define a template class or function that can be instantiated with different data types at compile time.

What is the difference between pass-by-value and pass-by-reference?

Pass-by-value involves passing a copy of the actual data to a function, while pass-by-reference involves passing a reference (memory address) of the data to a function. Pass-by-value creates a new copy of the data, while pass-by-reference allows the function to directly modify the original data.

What is a friend function?

A friend function in C++ is a function that is not a member of a class but has access to the private and protected members of that class. It is declared as a friend in the class where it is given access. Friend functions are often used for operator overloading or accessing private members without making them public.

Explain the concept of multiple inheritance.

Multiple inheritance in C++ occurs when a class inherits from more than one base class. It allows a derived class to inherit attributes and behaviors from multiple base classes. However, it can lead to the "diamond problem" where ambiguity arises if multiple base classes have the same member names. C++ provides virtual inheritance to resolve such issues.

What are smart pointers in C++?

Smart pointers in C++ are objects that manage the memory of dynamically allocated objects, ensuring proper memory deallocation when the smart pointer goes out of scope. They help prevent memory leaks and make memory management easier by automatically releasing memory when it's no longer needed. Common smart pointer types are shared_ptr, unique_ptr, and weak_ptr.

How does exception handling work in C++?

Exception handling in C++ allows you to handle runtime errors gracefully. When an exceptional situation occurs, you can use the "try-catch" block to catch and handle the exception. The "try" block contains the code that might throw an exception, and the "catch" block handles the exception by providing code to execute in response. If no catch block matches the exception, the program terminates.

What is Operator Overloading in C++?

Operator overloading in C++ allows you to define how operators behave for user-defined classes. It enables you to use operators like +, -, *, etc., with custom types.

Explain the Concept of Virtual Functions and Polymorphism:

Virtual functions enable polymorphism, allowing derived classes to provide their own implementation of a function defined in the base class. Polymorphism allows you to treat objects of derived classes as objects of the base class.

What is the Difference Between Overloading and Overriding?

Overloading involves defining multiple functions with the same name but different parameter lists in the same scope. Overriding is the ability of a derived class to provide a specific implementation of a function already defined in its base class.

What is a Virtual Destructor?

A virtual destructor is a destructor declared in a base class with the virtual keyword. It ensures that the appropriate derived class destructor is called when objects are deleted through a pointer to the base class.

Explain the Use of "const" Keyword in C++:

The const keyword is used to indicate that a variable or function parameter's value cannot be changed. It helps enforce data immutability and prevents unintended modifications.

What are Lambda Expressions in C++?

Lambda expressions are anonymous functions that can be defined inline. They are particularly useful for defining simple functions as arguments to algorithms and for capturing variables from the enclosing scope.

What is the Standard Template Library (STL) in C++?

The Standard Template Library (STL) is a collection of C++ template classes to provide general-purpose classes and functions with templates that implement many popular and commonly used algorithms and data structures like vectors, lists, and maps.

Explain the Concept of RAII (Resource Acquisition Is Initialization):

RAII is a programming technique in C++ where resource management is tied to object lifetime. Resources like memory, files, or network connections are acquired when an object is created and released when it's destroyed, ensuring proper resource management.

What are Move Semantics and Rvalue References?

Move semantics and rvalue references allow efficient transfer of ownership of resources from one object to another without unnecessary copying. They improve performance when dealing with temporaries and dynamic memory allocation.

Explain the Purpose of the "auto" Keyword:

The auto keyword is used for automatic type deduction during variable declaration. It allows the compiler to infer the variable's type based on the assigned value, simplifying code and improving readability.

Explain the PIMPL (Pointer to Implementation) Idiom:

The PIMPL idiom is a design pattern that hides the implementation details of a class behind a pointer. It helps minimize compilation dependencies and provides better encapsulation, allowing changes to the implementation without affecting the interface.

What are C++ Concepts?

C++ Concepts are a feature introduced in C++20 that provide a way to specify requirements for template parameters. They enable better error messages and ensure that templates are instantiated only with valid types that satisfy the specified constraints.

What are Custom Allocators in C++?

Custom allocators are user-defined memory allocation strategies that can be used with containers like vectors and maps. They allow you to control memory allocation and deallocation based on specific application requirements.

Explain the C++ Modules Feature:

C++ Modules is a feature introduced in C++20 that aims to improve the efficiency of the compilation process and eliminate the need for header files. Modules allow you to encapsulate and isolate declarations, making code organization and management more straightforward.

What is C++ Reflection?

C++ Reflection refers to the ability of a programming language to inspect and manipulate its own structure and metadata at runtime. C++ does not have built-in reflection, but various proposals and libraries aim to bring reflection-like capabilities to the language.

Explain the difference between "pre-increment" and "post-increment" operators.

The "pre-increment" (`++i`) operator increments the value before using it, while the "post-increment" (`i++`) operator uses the value and then increments it.

Swapping of Three Numbers Without Using Temporary Variable:


#include < iostream>
using namespace std;

int main() {
int a = 5, b = 10, c = 15;
cout << "Before swapping: a = " << a << ", b = " << b << ", c = " << c << endl;
a = a + b + c;
b = a - (b + c);
c = a - (b + c);
a = a - (b + c);

cout << "After swapping: a = " << a << ", b = " << b << ", c = " << c << endl;
return 0;
}

Reverse a String:

#include < iostream>
#include < string>
using namespace std;
int main() {
string input = "Hello, World!";
string reversed = "";
for (int i = input.length() - 1; i >= 0; i--) {
reversed += input[i];
}
cout << "Original String: " << input << endl;
cout << "Reversed String: " << reversed << endl;
return 0;
}

Factorial of a Number:


#include < iostream>
using namespace std;
int factorial(int n) {
if (n == 0 || n == 1)
return 1;
return n * factorial(n - 1);
}
int main() {
int n = 5;
cout << "Factorial of " << n << " is: " << factorial(n) << endl;
return 0;
}

Fibonacci Series:

#include < iostream>
using namespace std;
void fibonacci(int n) {
int a = 0, b = 1;
cout << "Fibonacci Series: ";
for (int i = 0; i < n; i++) {
cout << a << " ";
int sum = a + b;
a = b;
b = sum;
}
}
int main() {
int n = 10;
fibonacci(n);
return 0;
}

Explain the difference between "strcpy" and "memcpy".

Both functions copy memory, but `strcpy` is specific to strings and stops at the null terminator, while `memcpy` copies a specified number of bytes and isn't restricted to strings.

What is the "volatile" keyword used for?

The "volatile" keyword indicates that a variable's value may change unexpectedly, typically due to external factors, and should not be optimized by the compiler.

How can you find the length of a null-terminated string using a pointer?

You can iterate through the string using a pointer and count the characters until you reach the null terminator.

int stringLength(const char *str) {
int length = 0;
while (*str != '\0') { length++;
str++;
}
return length;
}
