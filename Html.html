What is HTML?

HTML stands for Hypertext Markup Language. It is the standard markup language used for creating and structuring content on the web. HTML uses various tags to define the structure, content, and presentation of a web page.

What are the basic building blocks of an HTML document?

An HTML document is composed of the following basic building blocks:

  • <!DOCTYPE html>: Specifies the document type and version of HTML.
  • <html>: The root element that encloses all other elements.
  • <head>: Contains metadata, links to external resources, and defines the document's structure.
  • <title>: Sets the title of the web page displayed in the browser's title bar.
  • <body>: Contains the visible content of the web page.

Explain the difference between HTML and HTML5.

HTML5 is the latest version of HTML and comes with several new features and improvements over previous versions of HTML. Some key differences include:

  • HTML5 introduces new semantic elements like <header>, <nav>, <article>, etc.
  • HTML5 supports multimedia elements like <audio> and <video>.
  • HTML5 introduces form input types like email, date, number, etc.
  • HTML5 includes APIs for client-side storage (localStorage and sessionStorage).
  • HTML5 introduces the <canvas> element for dynamic graphics and animations.

What is the purpose of the <head> element in an HTML document?

The <head> element is used to contain metadata and other non-visible information about the HTML document. It includes elements like <title> (sets the title of the page), <meta> (provides metadata like character encoding), and <link> (links to external resources like stylesheets).

Explain the concept of semantic HTML.

Semantic HTML refers to the practice of using HTML elements that convey meaningful information about the structure and content of a web page. Semantic elements like <header>, <nav>, <main>, <article>, <section>, and <footer> provide a clearer understanding of the content's purpose and enhance accessibility for users and search engines.

What are HTML attributes?

HTML attributes provide additional information about an HTML element. They are specified within the start tag of an element and are usually in the form of name-value pairs. For example, in the element <a href="https://www.example.com">Click Here</a>, href is the attribute name, and "https://www.example.com" is the attribute value.

What is the difference between inline and block-level elements?

Inline elements are those that do not start on a new line and only take up as much width as necessary. Examples include <span>, <a>, and <strong>. Block-level elements, on the other hand, start on a new line and occupy the full width of their parent container. Examples include <div>, <p>, and <h1>.

Explain the use of the <div> and <span> elements.

The <div> element is a block-level container used for grouping and styling sections of content. It is often used to create layout structures. The <span> element, on the other hand, is an inline container used to apply styles or scripts to a specific portion of text or content.

What is the purpose of the <img> tag?

The <img> tag is used to embed images into an HTML document. It has a src attribute that specifies the image's source URL and an alt attribute that provides a text description of the image for accessibility and in case the image cannot be loaded.

Explain the use of hyperlinks in HTML.

Hyperlinks, often referred to as links, are used to navigate between different web pages or resources. They are created using the <a> (anchor) element and the href attribute to specify the destination URL. Clicking on a hyperlink typically takes the user to the linked page or resource.

What is the difference between absolute and relative URLs?

An absolute URL specifies the complete address of a resource, including the protocol (e.g., http:// or https://). A relative URL, on the other hand, specifies the path to a resource relative to the current page's location. Relative URLs are often shorter and can simplify navigation within a website.

Explain the purpose of the <table> element.

The <table> element is used to create tabular data structures. It consists of one or more <tr> (table row) elements, each containing <td> (table data/cell) or <th> (table header/cell) elements. The <th> elements are used to define header cells in the table.

How can you create a hyperlink that opens an email client?

To create a hyperlink that opens an email client, you can use the <a> element with the href attribute set to mailto: followed by the email address. For example: <a href="mailto:example@example.com">Send Email</a>. Clicking the link will open the user's default email client with the recipient's address pre-filled.

What is the purpose of the <form> element?

The <form> element is used to create an HTML form that can be used to collect user input. It contains various form controls such as text fields, checkboxes, radio buttons, and buttons. The action attribute specifies the URL where the form data should be submitted, and the method attribute specifies the HTTP method (GET or POST) to use for the submission.

Explain the use of the <DOCTYPE> declaration in HTML.

The <DOCTYPE> declaration specifies the document type and version of HTML being used in the web page. It helps the browser understand how to interpret and render the page's content. For example, <!DOCTYPE html> indicates that the page uses HTML5.

What are semantic elements in HTML5?

Semantic elements in HTML5 are those that carry meaning about the structure and content of the web page. They provide better context to browsers, search engines, and developers. Examples of semantic elements include <header>, <nav>, <article>, <section>, and <footer>.

Explain the purpose of the <meta> tag in HTML.

The <meta> tag is used to provide metadata about the HTML document, such as character encoding, authorship, viewport settings for responsive design, and keywords for search engines. It is placed within the <head> section of the document and does not have a closing tag.

What is the difference between the <span> and <div> elements?

The <span> element is an inline container used to apply styles or scripts to a specific portion of text or content. The <div> element, on the other hand, is a block-level container used for grouping and styling larger sections of content. While <span> does not create line breaks, <div> starts on a new line and takes up the full width of its parent container.

Explain the concept of a self-closing tag in HTML.

A self-closing tag, also known as an empty tag or void element, is an HTML tag that does not require a closing tag because it doesn't enclose any content. Instead, it is written in a single tag format, usually ending with a forward slash before the closing angle bracket. For example: <br /> and <img src="image.jpg" />.

What is the purpose of the <iframe> element?

The <iframe> element is used to embed another HTML document or web page within the current page. It is often used to include external content like videos, maps, or advertisements. The src attribute specifies the URL of the external content to be displayed within the iframe.

Explain the use of the <figure> and <figcaption> elements.

The <figure> element is used to encapsulate media content like images, illustrations, diagrams, etc., along with an optional <figcaption> element that provides a caption or description for the media. This helps provide context and accessibility to users.

What is the purpose of the <pre> element?

The <pre> element is used to define preformatted text, such as code snippets or ASCII art, where whitespace and line breaks are preserved exactly as written. This is particularly useful for displaying code examples or other content that relies on specific formatting.

Explain the concept of the <select> element and its related elements.

The <select> element is used to create a dropdown menu or selection list. It contains one or more <option> elements that represent individual items in the list. The <optgroup> element can be used to group related <option> elements together. The <select> element can also include the <datalist> element, which provides suggestions as users type.

What is the role of the <meta name="viewport"> tag in HTML?

The <meta name="viewport"> tag is used to control the viewport settings for responsive design on mobile devices. It specifies the width and initial scale of the viewport, allowing the web page to adapt its layout to different screen sizes. For example: <meta name="viewport" content="width=device-width, initial-scale=1.0">.

Explain the purpose of the <canvas> element.

The <canvas> element is used to create graphics, animations, and interactive visualizations using JavaScript. It provides a drawing surface on which developers can use JavaScript to create complex shapes, images, animations, and games. The content within the <canvas> is rendered by JavaScript code.

What is the role of the <noscript> element?

The <noscript> element is used to provide alternative content that should be displayed when JavaScript is disabled in the user's browser. It is often used to show a message or instructions to users who don't have JavaScript enabled, encouraging them to enable it for a better experience.

Explain the use of the <header>, <nav>, and <main> elements in HTML5.

The <header> element is used to define the header section of a web page, typically containing a logo, site title, and navigation links. The <nav> element represents the navigation menu of the site. The <main> element encapsulates the main content of the page, excluding headers, footers, and sidebars.

What is the purpose of the <article> and <section> elements?

The <article> element is used to encapsulate a self-contained piece of content that could be distributed or reused independently, such as a news article, blog post, or forum post. The <section> element is used to group related content together and create a semantic structure within the document.

Explain the use of the <audio> and <video> elements.

The <audio> element is used to embed audio content on a web page. It supports various audio formats and provides controls for playing, pausing, and adjusting volume. The <video> element is used to embed video content, and it supports different video formats. It also provides controls for playing, pausing, seeking, and adjusting volume.

What is the purpose of the <details> and <summary> elements?

The <details> element is used to create a disclosure widget that can be clicked to reveal or hide additional content, such as additional information or options. The visible part of the widget is created using the <summary> element, which provides a label or title for the disclosed content.

Explain the use of the <progress> and <meter> elements.

The <progress> element is used to display the progress of a task or process, such as the completion of a download. It has attributes like value and max to indicate the current progress and maximum value. The <meter> element is used to display a scalar measurement within a known range, such as a battery charge level or temperature.

What is the purpose of the <abbr> and <cite> elements?

The <abbr> element is used to mark up abbreviations or acronyms in the content. It can optionally include a title attribute that provides the full expansion of the abbreviation. The <cite> element is used to mark up the title of a creative work, such as a book or movie, and can be used to provide citation information.

Explain the concept of HTML5 form validation.

HTML5 form validation allows you to validate user input without relying solely on JavaScript. It provides built-in validation attributes like required, pattern, min, max, and more, which are applied directly to form controls. This helps ensure that user input meets specific criteria before the form can be submitted, improving data quality and user experience.

What is the role of the <script> element in HTML?

The <script> element is used to include external JavaScript code or inline JavaScript code within an HTML document. It is placed either in the <head> section or just before the closing </body> tag. External scripts are linked using the src attribute, while inline scripts are written directly within the <script> tags.

Explain the difference between the <script> tag's async and defer attributes.

The async attribute in the <script> tag indicates that the script should be executed asynchronously, allowing the HTML parsing to continue while the script is being fetched. The script will be executed as soon as it's downloaded, which may lead to out-of-order execution. The defer attribute, on the other hand, also allows asynchronous execution but ensures that the script is executed in the order they appear in the document, after the document is parsed.

What is the purpose of the <link> element in HTML?

The <link> element is used to link external resources to an HTML document. It is commonly used to link stylesheets using the rel="stylesheet" attribute. The <link> element can also be used to link to other resources like icons and web fonts, and it can define relationships between documents using attributes like rel and hreflang.

Explain the use of the <base> element in HTML.

The <base> element specifies a base URL that is used for all relative URLs within the document. It helps browsers resolve relative URLs accurately. The <base> element's href attribute specifies the base URL. This is particularly useful when working with multiple linked resources or when hosting content on a different domain.

What is the purpose of the <time> element?

The <time> element is used to mark up a specific time or a range of time within the content. It helps convey the semantic meaning of time-related information to both users and machines. The <time> element can include attributes like datetime to provide machine-readable date and time information, and the content of the element is displayed to users.

Explain the concept of accessibility in web development.

Accessibility, often referred to as "a11y," refers to designing and developing web content and applications in a way that ensures people with disabilities can perceive, navigate, interact, and understand the content effectively. This includes considerations for users with visual, auditory, motor, and cognitive impairments. Web developers use semantic HTML, proper labeling, keyboard navigation, and other techniques to improve accessibility.

What is the role of the <wbr> element?

The <wbr> element, short for "word break opportunity," is used to indicate a point where a line break is suggested within a long word or a string of text. It allows browsers to break long words across lines without hyphenation. The <wbr> element is particularly useful for improving the readability of text in narrow columns or containers.

Explain the use of the <iframe sandbox> attribute.

The <iframe> element's sandbox attribute is used to apply a set of restrictions to the content displayed within the iframe. This helps mitigate security risks by isolating the content from the parent document. The sandbox attribute can include values like allow-same-origin, allow-scripts, and allow-forms, which define what actions the embedded content is allowed to perform.

What is the purpose of the <datalist> element?

The <datalist> element is used to provide a predefined list of options for an <input> element with the list attribute. It enhances user experience by offering auto-completion and suggestions as users type in an input field. The <option> elements within the <datalist> define the available options that users can choose from.

Explain the concept of responsive web design.

Responsive web design is an approach to designing and coding websites to provide an optimal viewing experience across a wide range of devices and screen sizes, including desktops, tablets, and mobile phones. It involves using fluid grids, flexible images, and media queries to adapt the layout and design of the website based on the capabilities and dimensions of the user's device.

What is the role of the <picture> element?

The <picture> element is used to provide multiple sources of an image and to allow the browser to choose the best source based on the user's device and screen size. It is often used in combination with the <source> element to offer different image formats and resolutions for various devices, helping improve performance and user experience.
